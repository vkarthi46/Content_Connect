# [Content-Connect](https://www.creative-tim.com/product/material-dashboard-pro-angular2)

## Quick start

Clone the Repo

## Terminal Commands

1. Install NodeJs from [NodeJs Official Page](https://nodejs.org/en).
2. Open Terminal
3. Go to your file project
4. Run in terminal: ```npm install```
5. And: ```npm start```
